# abx-hero (carpeta para /public)
Copiá esta carpeta **abx-hero/** dentro de `public/` de tu repo.

Estructura resultante:
- public/abx-hero/index.html
- public/abx-hero/styles.css
- public/abx-hero/main.js
- public/abx-hero/whatsapp-logo.png (opcional)
- public/abx-hero/models/base.glb

Probar: `https://TU-DOMINIO/abx-hero/`
Luego, en tu Home, embebé con:
<iframe src="/abx-hero/index.html" title="Abraxas Hero 3D" style="width:100%;height:100vh;border:0"></iframe>
